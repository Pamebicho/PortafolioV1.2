# Portfolio Profesional

Portafolio web desarrollado para presentar mi perfil como desarrolladora Full-Stack en formación, incluyendo proyectos, habilidades y experiencia.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap

## Estructura

El proyecto sigue una arquitectura por features:

- navbar
- hero
- about
- skills

## Objetivo

Construir una presencia profesional clara, moderna y funcional orientada a oportunidades laborales en el área TI.
